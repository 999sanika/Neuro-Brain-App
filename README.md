# Neuro Brain - Mental Health Consultation App

Neuro Brain is a mobile application UI designed for mental health support where users can connect with consultants.

## Features
- Client and Consultant selection
- Brain infographic information
- User login screen
- Mental health awareness interface
- Clean and modern UI design

## Screens
- Splash Screen
- Login Page
- Brain Information Screen
- Client / Consultant Selection
- Thank You Page

## Technologies Used
- Figma / UI Design
- Mobile UI/UX concepts

## Purpose
This project is designed to promote mental health awareness and provide a platform for consultation between users and mental health professionals.
